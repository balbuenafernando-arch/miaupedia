# Pendiente para producción

Requiere configuración o servicios externos:

- Crear el proyecto Supabase, ejecutar `supabase/schema.sql` y cargar datos peruanos reales.
- Sustituir las imágenes demo remotas por Supabase Storage y habilitar subida de fotos.
- Conectar las fichas demo a UUID reales para sincronizar reseñas y follows del directorio.
- Implementar interfaz completa de mensajería, panel empresarial, promociones y consola administrativa.
- Configurar trabajos programados para alertas de precio/reseñas/promociones.
- Añadir proveedor de correo, rate limiting perimetral, CAPTCHA y monitoreo.
- Incorporar catálogo oficial completo de departamentos, provincias y distritos.
